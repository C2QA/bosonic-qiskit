Wigner function as implemented in QuTiP (i.e., copy/paste). QuTiP is released under the BSD 3-clause license: https://github.com/qutip/qutip/blob/master/LICENSE.txt

See https://github.com/qutip/qutip/blob/master/qutip/wigner.py#L257-L300

Using an iterative method to evaluate the wigner functions for the Fock state |m >  < n|. The Wigner function is calculated as W = ∑mnρmnWmn where Wmn is the Wigner function for the density matrix |m >  < n|. In this implementation, for each row m, Wlist contains the Wigner functions Wlist = [0, ..., W_mm, ..., W_mN]. As soon as one W_mn Wigner function is calculated, the corresponding contribution is added to the total Wigner function, weighted by the corresponding element in the density matrix rhomn.
